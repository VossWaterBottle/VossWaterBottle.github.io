Project 5: Refactored Turtle Code

About the Project

For this project we refactored our original turtle code in order to be more efficient in creating graphics. We did this by creating functions and for loops for different aspects of the graphic. For example, instead of repeating single lines of code to command the turtle to create a graphic, you simply call a previously made function. This allows you to create much more detailed graphics as well as learning how to write efficient code with Python.

Sample Code


def draw_cloud(t, x, y, size=1.0):
    """Draw a cloud made of multiple overlapping circles"""
    jump_to(t, x, y)
    t.pencolor("white")

    # Draw multiple overlapping circles to form a cloud
    positions = [(0, 0), (size * 15, size * 10), (size * 30, 0),
                 (size * 45, size * 5), (size * 60, 0)]

    for pos_x, pos_y in positions:
        jump_to(t, x + pos_x, y + pos_y)
        draw_circle(t, size * 20, "white")
