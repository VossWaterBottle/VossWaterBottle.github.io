Project 7: Data Exploration

Within this project we looked at large data sets with vast columns and rows of data and used code to analyze this data. Within the code we made use of pandas, Matplotlib, and seaborn. With the help of these libraries I was able to analyse almost forty years of data! I looked into the fluctuation of oil barrel prices over the last two decades, interesting stuff I will say.
